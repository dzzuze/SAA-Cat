- **What was done:**

Today I started working on the Main Page, but before coding anything I wanted to understand how the page should actually look and feel.
I spent some time thinking about what sections the landing page should contain and how the user should move through the page. After a while I defined the main sections (of course with AI help):

- How It Works
- Topics

Then I recreated the layout in **Figma** so I could see everything visually before implementing it.

I tried different approaches: searching online and also generating some images that match the context of coding and studying.
It actually took longer than I expected, but now I have a few good options that fit the design.
By the end of the day I had more or less clear layout in Figma that I can use as a reference while implementing the page.

- **Decisions / Notes:**

Decided to:

- Design the page layout in Figma before writing code
- Split the page into sections that will later become React components
- Keep the structure simple and not overcomplicate it at the beginning
